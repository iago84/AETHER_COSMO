# Soporte y Ayuda

## ¿Dónde preguntar?
- Issues → dudas técnicas, bugs o solicitudes de features.
- Discusiones (si están activas) → ideas y debates.

## Antes de abrir un issue
- Revisa issues existentes.
- Adjunta versiones, logs y pasos de reproducción.

## Canales
- GitHub Issues del repositorio.
